# Many AWK

:monkey: manyawk is a AWK library that provides many comprehensive AWK utility functions. 

## Usage

You'll need to set AWKPATH to this library directory root when using manyawk.

```console
$ export AWKPATH=$PWD  # this library
$ echo $AWKPATH
/path/to/manyawk
```
